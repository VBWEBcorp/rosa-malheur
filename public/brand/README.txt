LOGOS ROSA MALHEUR
==================

Déposer ici les fichiers du logo (exactement ces noms) :

1) rosa-malheur.png         → version DÉTOURÉE (fond transparent)
                              Utilisée sur le site (hero, etc.).
                              C'est la plus importante.

2) rosa-malheur-blanc.png   → version FOND BLANC (optionnelle, pour impression/mails)

Astuce : exporter en PNG, largeur ~1200 px, fond transparent pour la version détourée.

Tant que rosa-malheur.png n'est pas présent, le site affiche automatiquement
un repli typographique « Rosa Malheur » (aucune image cassée).
